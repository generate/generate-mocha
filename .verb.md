## Quickstart

**Install**

Install [generate][] and `{%= name %}`:

```sh
$ npm install --global generate {%= name %}
```

**Generate a test.js file**

Initiate a prompt to generate a `test.js` file in the current working directory:

```sh
$ gen {%= alias %}
$ gen {%= alias %} --dest ./foo
```
